CyberShield-AI

⭐ Featured Final Year Project
The Brief

CyberShield-AI is a Flutter-based mobile cybersecurity tool — my Final Year Project at COMSATS. It unifies three real-world threats into one app: deepfake image detection using deep learning, spam SMS classification via NLP, and spam call detection. The goal was to give everyday users a single AI-powered shield against digital deception, all wrapped in a clean Flutter interface backed by Firebase for auth and real-time data.

FoodGuard AI

Hackathon Live on Vercel
The Brief

FoodGuard AI is a computer vision web app built during the GDG on Campus ATK AI/ML Fellowship hackathon. Upload a photo of any food item and the app uses Google's Gemini API to analyze visual cues — colour, texture, mould — and predict whether the food is fresh, expiring, or expired. I handled the frontend (HTML/CSS/JS) and Gemini API integration. The project is live on Vercel.
